Интернет-банк Промсвязьбанк 

 

1. Введите логин и пароль для входа в интернет-банк PSB-Retail.

 

 

 

2. В разделе «Оплата счетов» выберите нужный счет и нажмите «Оплатить счет…».

 

 

 

3. Выберите источник средств для оплаты счета.

 

 

 

4. Для подтверждения операции введите ключ из таблицы разовых ключей (если для работы в интернет-банке используется сертификат, то ввод ключа не потребуется).

 

 

 

5. Счет оплачен, сформировано платежное поручение и заявление на перевод.
